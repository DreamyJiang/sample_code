Tree painter
================

Paint tree to string or on the console. It is implemented in C++.

Like,

			┌─────┐
			│ROOT │
			└──┬──┘
		   ┌───┴───────┐
		 ┌─┴─┐       ┌─┴─┐
		 │16 │       │d13│
		 └─┬─┘       └─┬─┘
		┌──┴────┐     ┌┘
	┌───┴───┐ ┌─┴─┐ ┌─┴─┐
	│hello  │ │18 │ │j12│
	│world  │ └───┘ └───┘
	│a!     │
	└───────┘
